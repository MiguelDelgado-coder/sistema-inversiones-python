# 💰 Sistema de Inversiones en Python

Aplicación en consola que permite registrar inversiones, calcular ganancias (10%) y mostrar un resumen completo.

---

## 🚀 Funcionalidades

- Registro de múltiples inversiones
- Cálculo automático de ganancias
- Almacenamiento en listas
- Validación de datos (evita errores de entrada)
- Menú interactivo
- Resumen total de inversiones

---

## 🧠 Lógica aplicada

- Bucles `while`
- Condicionales `if / elif / else`
- Listas
- Acumuladores
- Validación con `try/except`

---

## ▶️ Cómo ejecutar

1. Tener Python instalado
2. Descargar el proyecto
3. Ejecutar:

```bash
python sistema_inversiones.py
